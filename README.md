# 🛒 TechStore Pro

A modern and responsive E-Commerce Website built using HTML, CSS, and JavaScript.

This project demonstrates frontend development concepts such as dynamic product rendering, product search, category filtering, shopping cart management, local storage integration, dark mode functionality, and responsive web design.

## 🌐 Live Demo

https://techstore-navin.netlify.app/

## ✨ Features

* Responsive User Interface
* Product Listing
* Product Search Functionality
* Category-Based Filtering
* Add to Cart Feature
* Cart Counter
* Local Storage Integration
* Dark Mode Toggle
* Mobile-Friendly Design
* Modern UI Components

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/bnavinkumar/E-commerce-website.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
